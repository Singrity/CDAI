Инструкция для Windows
### Перед запуском необходимо удостоверится установлен ли у вас python

1. Откройте меню пуск
2. Начните писать cmd
3. Откройте командную строку
4. Введите команду `python --version`
5. Если вывелась версия python значит интерпретатор установлен
6. Если нет, его необходимо установить из сайта `https://www.python.org/downloads/` рекомендую версию >= 3.10

### Далее необходимо открыть папку с проектом
нажмите на строку с адресом сверху и впишите `cmd`
далее откроется командная строка
Введите в открывашуюся командную строку команду `python launch.py` после чего должен открыться браузер с калькулятором
